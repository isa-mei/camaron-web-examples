# camaron-web-examples
The repository keep the examples for [camaron web](https://isa-mei.github.io/camaron-web/).

Camaron Web is a mesh visualizer that has been developed as part of several graduation projects for the Civil Engineering degree in Computer Science at Universidad de Chile. Professor [Nancy Hitschfeld Kahler](https://dcc.uchile.cl/nosotros/academico/nhitschfeld/) is the project's supervising professor. David Muñoz León (2018), Diego Zuñiga (2024), and Isabella Meirone (2026) are the students who have contributed to the project.

Camaron suports files in the format *.off, .node, .ele, .face, .poly, .smesh, .obj and .m3d*. The examples files are arranged in three categories: polygon mesh, polyhedron mesh and problematic mesh. The last one are files that Camaron accepts but for some reason can not be visualized.

The owner of this repository does not own the examples files. They were assembled to verify Camaron Web's functionality. While some were provided by Professor Nancy Hitschfeld, others were obtained online.

Some sources of this files are:
